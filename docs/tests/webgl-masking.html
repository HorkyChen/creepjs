检测逻辑说明

  • 使用 WebGL 绘制固定红色矩形。
  • 通过 readPixels 读取整块像素。
  • 对像素字节做 SHA-256 哈希。
  • 若哈希不等于内置期望值，则判定为 Masking detected。
